# 🚨 Raahi – Discreet Women’s Safety App (Fake Calculator UI)

## 👩‍💻 Team CodeCrafters

A smart and discreet women’s safety solution designed to send emergency alerts without alerting the attacker.

---

## 📌 Problem Statement

Women often cannot openly access safety apps during emergencies. Existing apps require visible interaction, which may not always be safe.

---

## 💡 Solution

**Raahi** is a safety app disguised as a calculator that allows users to trigger emergency alerts secretly.

* Entering a secret code (e.g., **991**) activates SOS.
* The app silently sends:

  * 📍 Live location
  * 🎥 Video
  * 🎙️ Audio
* Alerts are sent to **3 trusted contacts**.
* Entering **000** marks the emergency as resolved.
* Media is automatically saved to the user's device.

## 🗄️ Database Structure

1. **Users Table**

   * Stores user details

2. **Trusted Contacts Table**

   * Stores emergency contacts

3. **Emergency Alerts Table**

   * Tracks alert status (Active/Resolved)

---

## 🔄 Workflow

1. User opens calculator UI
2. Enters secret code (991)
3. SOS is triggered silently
4. Backend sends alerts + live data
5. Contacts receive real-time updates
6. User enters 000 → Emergency resolved
7. Recording saved to device
